Profile Card UI Design

1. This is a simple profile card created using HTML and CSS. It provides a minimalistic and elegant                 way to display basic information about an individual.
2. The profile card includes:
    -->  An Image
    -->  A Name
    -->  A Job Title
    -->  Social Media handles
    -->  Subscribe and Message buttons
    -->  Some Analytics such as  Likes, Shares, Messages  


3. Technologies Used
    -->  HTML
    -->  CSS

4. To Access the project You just need to: 
    step 1 --> Clone the repository or Download the .zip file.
    step 2 --> Extract the .zip file in your system.
    step 3 --> Open the project folder and open the 'index.html' in any browser.


The card will display the provided sample information. You can customize the content to make your own profile.
